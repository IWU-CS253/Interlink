# InterLink

A comprehensive sports league management system built with Flask that allows users to create leagues, form teams, schedule matches, and track standings across multiple sports.

## Features

- **League Management**: Create and manage leagues across different sports with customizable settings
- **Team Organization**: Create teams, manage rosters, and recruit players
- **Automated Scheduling**: Generate complete match schedules with customizable start dates and games per week
- **Score Tracking**: Submit and edit game results with real-time standings updates
- **User Authentication**: Secure login system with role-based access (Admin/Player)
- **Admin Controls**: Comprehensive admin dashboard for managing leagues, teams, schedules, and scores
